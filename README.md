README – Projekt 4: Strona Portfolio (HTML / CSS / Tailwind)
1. Opis projektu
Projekt przedstawia responsywną stronę internetową typu portfolio, stworzoną w technologiach HTML, CSS oraz TailwindCSS.
Strona prezentuje autora, jego prace oraz umożliwia kontakt poprzez dedykowaną podstronę.
Projekt jest w pełni statyczny, zoptymalizowany pod kątem wydajności i zgodny z zasadami mobile‑first.

Strona składa się z kilku podstron:
index.html – strona główna
about.html – sekcja o autorze
gallery.html – galeria prac
contact.html – formularz kontaktowy

2. Technologie użyte w projekcie
